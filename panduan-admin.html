00 — PENGANTAR

Panduan Operasional
Admin MeetSpace

Panduan lengkap untuk administrator dalam mengelola ruangan meeting dan mengkonfirmasi permintaan booking dari pengguna.

Selamat Datang, Admin! 👋

Sebagai Admin MeetSpace, Anda memiliki kendali penuh atas seluruh operasional sistem — mulai dari menambahkan ruangan baru, mengatur slot waktu, hingga mengkonfirmasi atau menolak permintaan booking dari karyawan.

👤 Pengguna Biasa — hanya bisa booking
🔑 Admin — kelola semua fitur
🏛️

Kelola Ruangan

Tambah, edit, nonaktifkan ruangan, dan atur fasilitas yang tersedia di setiap ruangan.

✅

Konfirmasi Booking

Review permintaan booking masuk, konfirmasi yang valid, dan tolak yang tidak sesuai.

📊

Sinkronisasi Data

Semua perubahan otomatis tersinkron ke file spreadsheet meetspace-database.xlsx.

📅

Kelola Kalender

Lihat semua booking dalam tampilan kalender bulanan dan atur jadwal blokir ruangan.

ℹ️
Versi Panduan Panduan ini berlaku untuk MeetSpace versi 1.0 — Mobile Friendly Edition. Semua tampilan layar dalam panduan ini merupakan simulasi antarmuka admin.
BAGIAN 1 — AKSES SISTEM
01 — AKSES

Mengakses Admin Panel

Admin Panel adalah halaman khusus yang hanya bisa diakses oleh pengguna dengan hak akses Administrator.

1

Buka Website MeetSpace

Akses melalui browser di perangkat apapun

Buka browser (Chrome, Edge, atau Firefox) lalu buka file meeting-booking-mobile.html atau URL yang sudah di-deploy. Pastikan koneksi internet aktif untuk memuat font.

2

Masuk ke Halaman Profil

Klik ikon 👤 di navigasi bawah atau sidebar

Di bagian navigasi bawah (mobile) atau menu atas (desktop), klik tab Profil. Ini akan membuka halaman profil pengguna Anda.

3

Akses Menu Admin

Scroll ke bawah halaman profil untuk menemukan menu Admin

Pada halaman Profil, scroll ke bawah. Pengguna dengan role Administrator akan melihat kartu khusus "🔑 Panel Admin" yang tidak terlihat oleh pengguna biasa. Klik kartu tersebut.

⚠️
Keamanan: Jika Anda tidak melihat menu Admin, berarti akun Anda belum diberi hak Admin. Hubungi superadmin atau IT untuk meminta akses.

03 — EDIT & NONAKTIFKAN

Edit & Nonaktifkan Ruangan

Cara mengubah informasi ruangan yang sudah ada atau menonaktifkannya sementara/permanen.

1

Temukan Ruangan yang Akan Diedit

Gunakan fitur pencarian atau filter di daftar ruangan

Di menu 🏛️ Ruangan, gunakan kolom pencarian atau filter berdasarkan lantai untuk menemukan ruangan. Klik baris ruangan yang ingin diedit untuk memperluas detailnya.

2

Klik Tombol Edit

Tersedia di kolom Aksi pada setiap baris ruangan

Di kolom paling kanan setiap baris ruangan, terdapat tombol aksi:

TombolFungsi
✏️ EditBuka formulir edit — ubah nama, kapasitas, fasilitas, dll.
⏸️ NonaktifkanSembunyikan ruangan dari pengguna (sementara, misal: renovasi)
🗑️ HapusHapus permanen dari sistem (hati-hati, tidak bisa dibatalkan!)
🚨
Perhatian — Hapus Permanen: Tombol Hapus akan menghapus ruangan dan SEMUA data terkaitnya secara permanen. Jika hanya ingin menutup ruangan sementara, gunakan Nonaktifkan saja.
3

Simpan Perubahan

Konfirmasi dan simpan hasil edit

Setelah mengedit, klik "💾 Simpan Perubahan". Sistem akan memberikan notifikasi konfirmasi dan data langsung terupdate di spreadsheet pada sheet 🏛️ Data Ruangan.

⚠️
Booking yang Sudah Ada: Jika ada booking aktif untuk ruangan yang dinonaktifkan, sistem akan memberikan peringatan. Admin perlu menghubungi pemesan dan menawarkan ruangan alternatif sebelum menonaktifkan.
04 — SLOT WAKTU

Mengelola Slot Waktu

Atur jam operasional ruangan dan blokir slot waktu tertentu agar tidak bisa dipesan.

1

Atur Jam Operasional Ruangan

Tentukan jam buka dan tutup setiap ruangan

Di halaman edit ruangan, geser ke bagian "⏰ Jam Operasional". Atur jam buka (misalnya 07:00) dan jam tutup (misalnya 20:00). Slot waktu yang muncul di halaman pengguna akan otomatis menyesuaikan rentang ini.

2

Blokir Slot Waktu Tertentu

Untuk pemeliharaan, acara khusus, atau hari libur

Buka menu 📅 Kalender di Admin Panel. Pilih ruangan dan klik tanggal yang ingin diblokir. Klik "🔒 Blokir Slot" lalu pilih jam yang ingin dikunci. Tambahkan keterangan alasan blokir.

Blokir hari libur nasionalPastikan slot otomatis terblokir di semua ruangan pada hari libur
Blokir jam maintenance rutinContoh: setiap Senin 07:00–08:00 untuk pembersihan ruangan
Blokir untuk acara khususReservasi prioritas untuk rapat direksi atau event perusahaan

06 — KONFIRMASI BOOKING

Mengkonfirmasi Booking

Proses review dan konfirmasi permintaan booking yang masuk dari pengguna. Ini adalah tugas utama Admin setiap harinya.

ALUR KONFIRMASI BOOKING

1

Buka Detail Booking yang "Menunggu"

Klik kode booking atau tombol Detail untuk melihat informasi lengkap

Di tabel booking, klik baris booking berstatus ⏳ Menunggu. Sebuah panel detail akan terbuka di sebelah kanan (desktop) atau muncul sebagai bottom sheet (mobile).

Panel detail menampilkan informasi lengkap:

  1. Identitas PemesanNama, departemen, email, dan nomor HP pengguna yang melakukan booking
  2. Detail RuanganNama ruangan, lantai, kapasitas, dan fasilitas yang dipesan
  3. Waktu BookingTanggal, jam mulai, jam selesai, dan total durasi
  4. Agenda MeetingTujuan dan deskripsi rapat yang diisi pengguna saat booking
  5. Jumlah PesertaPastikan tidak melebihi kapasitas maksimum ruangan
2

Periksa Kelayakan Booking

Gunakan checklist ini sebelum mengkonfirmasi

Sebelum mengklik Konfirmasi, periksa hal-hal berikut:

Tidak ada double-bookingPastikan ruangan tidak sudah dipesan pada slot waktu yang sama
Kapasitas sesuaiJumlah peserta tidak melebihi kapasitas maksimum ruangan
Ruangan dalam kondisi aktifStatus ruangan adalah Aktif dan tidak sedang dalam perawatan
Data pemesan lengkapNama, email, dan nomor HP pemesan sudah terisi dengan benar
Agenda meeting jelasAda keterangan agenda meeting yang terisi (bukan kosong)
3

Klik Tombol Konfirmasi

Ubah status dari "Menunggu" menjadi "Dikonfirmasi"

Setelah semua checklist terpenuhi, klik tombol hijau "✅ Konfirmasi Booking" di panel detail atau langsung dari kolom Aksi pada tabel.

✅
Notifikasi Otomatis: Begitu Admin mengklik Konfirmasi, sistem secara otomatis mengirim notifikasi email ke pemesan berisi: kode booking, nama ruangan, tanggal & waktu, dan QR code untuk akses ruangan.

Status booking di spreadsheet (📋 Data Booking) juga langsung berubah menjadi "Dikonfirmasi" dan formula dashboard akan terupdate secara real-time.

07 — TOLAK & BATALKAN

Menolak & Membatalkan Booking

Panduan untuk menolak permintaan yang tidak memenuhi syarat, atau membatalkan booking yang sudah dikonfirmasi.

1

Menolak Permintaan Booking Baru

Untuk booking berstatus "Menunggu" yang tidak bisa dikonfirmasi

Dari panel detail booking, klik tombol merah "❌ Tolak Booking". Sebuah dialog akan muncul meminta alasan penolakan. Pilih alasan dari daftar atau tulis manual:

Alasan PenolakanKapan Digunakan
🔄 Double bookingRuangan sudah dipesan di slot yang sama
📏 Kapasitas melebihiPeserta melebihi kapasitas ruangan
🔧 Ruangan dalam perawatanAda perbaikan atau maintenance terjadwal
📋 Data tidak lengkapInformasi pemesan atau agenda tidak jelas
✍️ Alasan lainnyaTulis alasan custom secara manual
💬
Etika Admin: Selalu isi alasan penolakan dengan jelas dan sopan. Alasan ini akan dikirim ke pemesan melalui email sehingga mereka bisa melakukan perbaikan atau mencari alternatif.
2

Membatalkan Booking yang Sudah Dikonfirmasi

Situasi darurat: ruangan rusak, bencana, atau perubahan mendadak

Untuk booking yang sudah berstatus ✓ Dikonfirmasi, klik "🔄 Batalkan Booking". Sistem akan meminta konfirmasi dua kali karena ini tindakan berisiko tinggi.

🚨
Penting: Membatalkan booking yang sudah dikonfirmasi harus disertai pemberitahuan aktif ke pemesan. Sistem mengirim email otomatis, tapi Admin juga disarankan menghubungi langsung via telepon atau WhatsApp untuk situasi mendadak.
BAGIAN 4 — SPREADSHEET & DATA
08 — SPREADSHEET

Sinkronisasi Data ke Spreadsheet

Semua data website tersimpan dan tersinkron ke file meetspace-database.xlsx. Berikut cara membaca dan mengelolanya.

1

Struktur File Spreadsheet

File Excel terdiri dari 4 sheet yang saling terhubung

SheetIsi DataDiupdate Oleh
📊 Dashboard KPI, statistik per ruangan, grafik ringkasan Otomatis (formula)
📋 Data Booking Semua record booking: kode, ruangan, tanggal, status, pemesan Admin & Sistem
👤 Profil Pengguna Data karyawan, statistik booking pribadi Pengguna & Admin
🏛️ Data Ruangan Master data ruangan: kapasitas, fasilitas, status Admin
2

Cara Update Manual di Spreadsheet

Ketika perlu mengoreksi data langsung dari Excel

Buka file meetspace-database.xlsx di Microsoft Excel atau Google Sheets. Untuk mengkonfirmasi booking secara manual di spreadsheet:

  1. Buka sheet "📋 Data Booking"Temukan baris booking yang berstatus "Menunggu" di kolom G (Status)
  2. Klik sel Status (kolom G)Ubah teks "Menunggu" menjadi "Dikonfirmasi" atau "Dibatalkan"
  3. Simpan file (Ctrl+S)Perubahan akan otomatis terupdate di sheet Dashboard karena formula terhubung
  4. Sinkronkan ke websiteDi Admin Panel website, klik "🔄 Sinkronisasi Manual" agar perubahan tampil di halaman pengguna
⚠️
Catatan Penting: Perubahan langsung di file Excel tidak otomatis tampil di website. Selalu gunakan tombol "Sinkronisasi Manual" di Admin Panel setelah mengedit spreadsheet secara langsung.
3

Tambah Ruangan via Spreadsheet

Cara cepat menambahkan banyak ruangan sekaligus

Buka sheet "🏛️ Data Ruangan". Tambahkan baris baru di bawah data yang ada dengan mengisi kolom:

TugasCara MelakukannyaFrekuensi
